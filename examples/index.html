Curi Examples

These are some Curi example projects that you can use for reference while building your own application. Most of these examples have CodeSandbox demos embedded with them, but some do not. Each example includes source code available through the Curi package on GitHub.

  • Active Links

    Style links when they match the current location

  • Authentication

    Automatically redirect to a login page when attempting to access private content

  • Basic Vue

    A simple Curi app rendered using VueJS

  • Basic Svelte

    A simple Curi app rendered using Svelte

  • Blocking Navigation (React)

    Prevent navigation away from a half-filled form

  • Blocking Navigation (Vue)

    Prevent navigation away from a half-filled form

  • Breadcrumbs (React)

    Render breadcrumb links to ancestor routes

  • Breadcrumbs (Vue)

    Render breadcrumb links to ancestor routes

  • Code Splitting

    Use import() to enable Webpack code splitting

  • Data Loading

    Display a loading bar while waiting for data to load

  • Modal Routes

    Load a route in a modal (the Pinterest model)

  • Redux

    Integrate Redux, React, and Curi (easily!)

  • Script Tags

    Load Curi packages using script tags instead of a bundle

  • Server Rendering

    Render your application on the server using Node (this example uses Express)

  • Side Effects

    Add side effects that always respond to navigation

  • Transitions

    Transition between routes using react-transition-group

You can see the source code for all of the examples on GitHub.
