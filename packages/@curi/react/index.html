Installation

You can install the react package from NPM.

npm install @curi/react

If you prefer to use script tags, Unpkg will always have the latest version of react available for you.

<script src="https://unpkg.com/@curi/react@1.0.0-beta.5/dist/curi-react.js" />

There is also a minimized version available if you change the filename to react.min.js

// the package will be attached to the window
window.CuriReact

API

import {
  Navigator,
  Link,
  Redirect,
  Block,
  Active,
  curious
} from '@curi/react';
