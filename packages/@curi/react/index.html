About#

The @curi/react package just re-exports a number of React specific Curi packages. You can read the documentation for each one on their respective pages.

  • react-navigator
  • react-link
  • react-redirect
  • react-block
  • react-active
  • react-curious

This is convenient, but if you want to minimize your application's bundle size (through tree shaking), but should import from the actual packages that these components come from.

Installation#

You can install the react package from NPM.

npm install @curi/react

If you prefer to use script tags, Unpkg will always have the latest version of react available for you.

<script src="https://unpkg.com/@curi/react@undefined/dist/curi-react.js" />

There is also a minimized version available if you change the filename to react.min.js. The package will be attached to the window as window.CuriReact.

API#

import {
  Navigator,
  Link,
  Redirect,
  Block,
  Active,
  curious
} from '@curi/react';
