@curi/side-effect-title

v1.0.0-beta.5
GitHub RepoNPM Package

About#

This package allows you to set title properties on your routes and will automatically set the page's title whenever the location changes.

Installation#

You can install the side-effect-title package from NPM.

npm install @curi/side-effect-title

If you prefer to use script tags, Unpkg will always have the latest version of side-effect-title available for you.

<script src="https://unpkg.com/@curi/side-effect-title@1.0.0-beta.5/dist/curi-side-effect-title.js" />

There is also a minimized version available if you change the filename to side-effect-title.min.js. The package will be attached to the window as window.CuriSideEffectTitle.

API#

createTitleSideEffect#

import createConfig from '@curi/core';
import createTitleSideEffect from '@curi/side-effect-title';

const setTitle = createTitleSideEffect({ suffix: '| My Site' });

const config = createConfig(history, routes, {
  sideEffects: [setTitle]
});

In order for this to work, you will need to set title properties on your routes. You can learn more about route.title in the all about routes guide.

You can provide a prefix and/or a suffix string that will be included before/after the title.

const prefixedTitle = createTitleSideEffect({ prefix: 'Before |'});
// response.title = 'Middle'
// document.title = 'Before | Middle';

const suffixedTitle = createTitleSideEffect({ suffix: '| After'});
// response.title = 'Middle'
// document.title = 'Middle | After';
